## How to Download

You can download these images using a browser or command line tools:

### Using wget (in terminal):

```bash
cd public/img
wget -O coffee-hero-bg.jpg "URL_HERE"
```

### Using curl (in terminal):

```bash
cd public/img
curl -o coffee-hero-bg.jpg "URL_HERE"
```

### Or simply:

Open each URL in your browser and save the image with the corresponding filename.
